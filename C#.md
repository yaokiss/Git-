# C#

### dos命令

dir 列出当前目录下的文件及目录 directory

md  创建目录  make directory

cd  进入指定目录 change directory

cd.. 退回到上一级目录

cd\ 直接回到根目录

rd  删除目录（空目录） remove directory

del 删除文件

rmdir /s/q xxx  删除整个目录和目录下所有的目录和文件（/s 是代表删除所有子目录和文件，/q 是不要在删除是提示是否删除）

rmdir /s/q d:/xxx/yyy 删除指定磁盘的指定目录

exit 退出dos命令行

help 寻求帮助

ipconfig 获取当前主机的IP
